**Thesis.** A triptych. Utopia — a final, authoritative social arrangement — is structurally incoherent under agent-relative value ("ill-typed"): valuation is an activity internal to agents, so there is no common unit and no default aggregation, and arrangements claiming finality must degrade agency to survive value drift (value freezing, value policing, exit suppression — "a machine for degrading agency"). Civilizational optimization is utopia in dynamic form: scalar collapse of plural futures licenses sacrifice, refinement of the metric only increases sacrifice's efficiency, and even bare survival constraints authorize override — the missing option in Hanson's trilemma is structural restraint. The Sacrifice Pattern is the mechanism both run on: whenever systems optimize under standing asymmetry, agency loss becomes a control variable — no malice required; dashboards and PR now serve the function once served by theology.
