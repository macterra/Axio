**Thesis.** An agent is any system with predictive modeling, counterfactual reasoning, and causal efficacy; a mind is the reflective informational subsystem within one — agents can exist without minds, never the reverse. The ladder: function → program → recursive program → simulation → mind, a recursive simulation of agency, substrate-independent in principle.
