**Thesis.** The defensible core is five architectural claims — authority separable from intelligence; deterministic replay with stochastic models; laundering constrained at the kernel layer; reflection without privilege; explicit, evaluable succession — and mistaking that internal coherence for solved alignment is precisely the failure mode the program built its own defenses against. The kernel enforces syntax, not meaning; the translation boundary remains a vulnerability; replayability is inspectability, not correctness. Structural governance does not save the surrounding world: failure-free governance is never observed, help selects failure modes, peers partition rather than harmonize. And the epistemic discipline is itself load-bearing: coherence is not constraint, LLM-assisted convergence is not evidence, and if the framework ever feels obviously correct, it has already ceased to function scientifically.
