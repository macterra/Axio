**Thesis.** Attaching a moral claim to a specified agent converts it into an empirically evaluable statement: 'X is wrong' is a naked preference, 'given agent A's values, X is wrong' is conditionally objective. Agent-binding is neither Mackie's error theory nor Joyce's fictionalism, and it is emphatically not relativism — the standard is explicit, so claims remain criticizable.
