# Volume edit records

These internal records document consequential editorial decisions made during the chapter-level volume passes. The publication build excludes `book/editorial/`, so the notes remain beside the manuscript without appearing in the book.

Each record should identify:

- structural and argumentative changes;
- claims whose calibration changed;
- unresolved derivations or author decisions;
- intentional deviations from the architecture memo;
- validation performed before review.
