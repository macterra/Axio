**Thesis.** Interpretation always presupposes prior interpretation, so a deliberately circular minimal ontology (Property→Object→System→Process→Agent) is a virtue, not a defect — and against Hoffman's interface theory, perception is a partial but structurally anchored slice of reality, not a useful fiction.
